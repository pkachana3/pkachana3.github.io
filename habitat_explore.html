003 : Map Exploration and Object Instance Search

Algorithmic approaches to finding object instances in a pre-mapped environment, simulated using Habitat-lab.

Faculty Advisor: Jacob Abernathy

Abstract: Efficient exploration of a pre-computed map has many use cases, such as in warehouse robotics, household robotics, and building security systems. The ability to plan robust and copmrehensive paths to observe a environment can open up door to many interesting robotic applications. It can be extended to automated transportation, objects search, and anomoly detection, for example. This study finds and benchmarks traditional approaches to map exploration and extends them to robotic road-mapping and reinforcement learning domains.
